# Phishing Email Detection Model

## Overview
The Phishing Email Detection Model is a Python-based machine learning project that classifies emails as **Phishing or Safe** based on their text content and URL patterns. The project demonstrates basic Natural Language Processing (NLP) and supervised learning techniques for cybersecurity applications.

## Features
- Detects phishing and legitimate emails
- Extracts email text features using TF-IDF
- Analyzes URLs in email content
- Counts special characters for pattern detection
- Classifies emails using Logistic Regression
- Displays accuracy score and confusion matrix

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLP (TF-IDF Vectorization)
